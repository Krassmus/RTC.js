Rtc.js

A webRTC library that should make an webRTC as easy as jQuery.ajax.

API of RTC.js

RTC.Connection

RTC.Connection is a class so that you can establish a connection by calling 

var connection = new RTC.Connection({
    'myvideo': document.getElementById("#me_smiling_in_the_cam"),
    'video': document.getElementById("#remoteuser_smiling_in_the_cam"),
    'sendOffer': function (offer) {
        //and here we send the offer right sto our signalling server
        jQuery.ajax({
            'url': "https://myserver/signaling/start",
            'data': {
                'offer_sdp': offer
            }
        });
    }
});

The following tables describe all options

option effect default
sendOffer This function is called to start signaling. The first and only parameter is the SDP-string and it is up to YOU to deliver that to the other side of the connection. For each connection you should either define sendOffer or offer.
